Comparing classifiers using R

About the project

Summary:

In this project, I undertake an analysis of the renowned Titanic dataset, employing the R programming language and RStudio software. The methodology includes the following steps:
- Conduct an exploratory analysis. This dataset includes various attributes of the Titanic's passengers as well as their survival status.
- Preprocess the data. The preprocessing steps include the removal of unnecessary columns, handling of missing values, one-hot encoding of categorical attributes, transformation of the 'gender' attribute to a numeric form, and normalization (scaling) of the data, given that many algorithms are sensitive to the magnitude of attribute values. Additionally, Principal Component Analysis (PCA) is employed for dimensionality reduction, which not only enhances computational efficiency but also improves the performance of several algorithms.
- Compare seven renowned classification algorithms. This involves training these classification algorithms using the K-fold cross-validation ensemble method to predict passenger survival based on their attributes. The prediction results of these classifiers are subsequently evaluated using five different metrics (Accuracy, AUC score, Recall, Precision, F1-score) and compared. Finally, the most suitable classifier is selected.

Tools used:

R-studio with the following packages: class, pROC, e1071, randomForest, rpart, neuralnet, stats.

Project details:

Classification is among the most prevalent applications of machine learning. It involves the process of assigning an object or record to a specific class based on its attributes. To enable the machine to accurately determine the appropriate class for a given object, it must first be trained using labeled examples.

Numerous algorithms are dedicated to the task of classification. In this project we will compare seven of them:
1) K-Nearest Neighbors (KNN)
2) Artificial Neural Networks (ANN)
3) Support Vector Machines (SVM)
4) Random Forests
5) Naive Bayes classifier
6) Decision Tree
7) Logistic Regression

Firstly an exploratory analysis is being made to the dataset and its features. Inside the code (which you can view below in the "R-code" section) there are comments and discussion about each step of the procedure and the analysis from the beginning to the end. Secondly a data preprocessing step is performed in order for the dataset to be in appropriate format for the algorithms to utilize it. Finally the classifiers are trained and tested and their predictions are measured with five different metrics.

Tools - Packages

R-Studio was used along with the following libraries-packages: class, pROC, e1071, randomForest, rpart, neuralnet, stats.

  • The "class" library is primarily used for classification, and it includes functions for K-Nearest Neighbors (KNN) classification. One of the most commonly used functions from the class package is knn(), which performs K-Nearest Neighbors classification.

  • The "pROC" library is used to compute the area under the ROC curve (AUC). The AUC-score is a single scalar value summarizing the performance of the classifier. An AUC value of 1 indicates perfect classification, while a value of 0.5 suggests no better than random guessing.

  • The "e1071" is a comprehensive package that includes a variety of tools for machine learning, statistical computing, and data analysis. In this project we use it for the implementation of Support Vector Machines (SVM) and Naive Bayes classification algorithms. The svm() function allows us to fit SVM models and the naiveBayes() function allows us to implement the Naive Bayes method.

  • The "randomForest" library is used for implementing the Random Forest algorithm, which is an ensemble learning method for classification, regression, and other tasks. Random Forest operates by constructing multiple decision trees during training and outputting the mode of the classes (classification) or mean prediction (regression) of the individual trees.

  • The "rpart" library is used for recursive partitioning and regression trees. It provides functions to create classification and regression models in the form of decision trees. These trees can be used for both predictive modeling and exploratory data analysis.

  • The "neuralnet" library is used for training and evaluating artificial neural networks (ANNs). It is designed to provide a flexible and straightforward way to create and train neural networks for regression and classification tasks. It supports various customization options, including specifying the number of hidden layers and neurons, choosing the activation functions, and setting learning parameters.

  • The "stats" package is preinstalled with the base R installation, so no extra installation is needed. This package includes a variety of functions for statistical analysis, including the glm() function which we use for fitting generalized linear models such as Logistic Regression.
